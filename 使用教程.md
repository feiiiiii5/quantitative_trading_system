# QuantSystem Pro 量化交易系统使用教程

## 1. 系统简介

QuantSystem Pro 是一套功能强大、高效、易用的专业级量化交易系统，集成了业界领先的量化策略、高性能回测引擎和完善的风险管理体系。系统采用模块化设计，易于扩展，适合从入门到专业的各级用户。

### 核心特性

- 🚀 **高性能**: 向量化回测引擎，支持多策略并行计算
- 🧠 **AI驱动**: 集成机器学习预测模型（XGBoost、LightGBM）
- 📊 **丰富策略**: 多因子模型、自适应市场、统计套利等
- 🛡️ **完善风控**: VaR、CVaR、压力测试、动态止损
- 💰 **专业资金管理**: 凯利公式、波动率目标、风险平价
- 📈 **交互界面**: Streamlit Web界面，可视化分析

## 3. 快速开始

### 3.1 回测示例

使用均线交叉策略回测上证指数（000001）：

```bash
# 标准版入口
python main.py backtest --symbol 000001 --strategy ma_cross

# 专业版入口（支持更多功能）
python pro_main.py backtest --symbol 000001 --strategy ma_cross
```

### 3.2 策略分析

运行完整的因子分析和绩效归因：

```bash
python pro_main.py analyze --symbol 000001
```

### 3.3 参数优化

对均线交叉策略进行参数优化：

```bash
python main.py optimize --symbol 000001 --strategy ma_cross
```

### 3.4 Walkforward分析

进行跨周期验证，评估策略的稳健性：

```bash
python main.py walkforward --symbol 000001 --strategy ma_cross --windows 5
```

### 3.5 生成报告

回测并生成详细的HTML报告：

```bash
python pro_main.py backtest --symbol 000001 --strategy ma_cross --report report.html
```

## 5. 市场支持

系统支持三个主要市场，会自动识别股票代码所属市场：

### 5.1 A股市场（CN）
- 代码格式：6位数字（如000001、600519）
- 交易规则：T+1结算，±10%涨跌停
- 费用：佣金+印花税（卖出时）+过户费

### 5.2 港股市场（HK）
- 代码格式：5位数字（如0700、9988）
- 交易规则：T+2结算，无涨跌停限制
- 费用：佣金+印花税

### 5.3 美股市场（US）
- 代码格式：字母（如AAPL、MSFT）
- 交易规则：T+1结算，无涨跌停（但有熔断机制）
- 费用：多数券商零佣金，SEC费（卖出时）

## 6. 高级功能

### 6.1 自定义策略

您可以通过继承 `BaseStrategy` 类来创建自定义策略：

```python
from strategies.base_strategy import BaseStrategy

class MyStrategy(BaseStrategy):
    def __init__(self, parameters=None):
        super().__init__(name="MyStrategy", parameters=parameters)
    
    def init(self):
        # 初始化计算
        self.sma = self.data.close.rolling(20).mean()
        self.data.add_indicator('sma', self.sma)
    
    def next(self, index: int):
        # 交易逻辑
        if self.data.close.iloc[index] > self.sma.iloc[index]:
            return self.buy(quantity=100, reason="价格突破均线")
        elif self.data.close.iloc[index] < self.sma.iloc[index]:
            return self.sell(quantity=100, reason="价格跌破均线")
        return None
```

### 6.2 配置管理

系统配置位于 `config/settings.py`，您可以根据需要修改：

- **数据源配置**：选择 akshare、tushare 或 baostock
- **风控配置**：设置最大仓位、止损、止盈等参数
- **交易配置**：设置佣金、滑点、初始资金等
- **市场配置**：各市场的交易规则和费用

### 6.3 性能优化

- **向量化回测**：使用 `--mode vectorized` 参数启用向量化回测，速度提升100倍以上
- **数据缓存**：系统会自动缓存数据，减少重复获取
- **并行计算**：支持多策略并行回测

## 7. 风险管理

系统内置了完善的风险管理功能：

### 7.1 风险指标
- **VaR / CVaR**：风险价值与条件风险价值
- **夏普比率**：风险调整收益
- **最大回撤**：历史最大回撤及持续期
- **Calmar比率**：年化收益/最大回撤

### 7.2 风控机制
- **动态仓位管理**：基于凯利公式的仓位计算
- **波动率目标**：根据波动率调整仓位
- **追踪止损**：自动追踪止损
- **压力测试**：模拟极端市场情况
- **多市场状态适配**：根据市场状态调整策略参数

## 9. 常见问题

### 9.1 数据获取失败
- 检查网络连接
- 尝试切换数据源（在 config/settings.py 中修改）
- 确认股票代码是否正确

### 9.2 策略表现不佳
- 尝试参数优化：`python main.py optimize --symbol 000001 --strategy ma_cross`
- 进行Walkforward分析：`python main.py walkforward --symbol 000001 --strategy ma_cross`
- 考虑使用更复杂的策略，如 `multi_factor` 或 `adaptive`

### 9.3 回测速度慢
- 使用向量化回测：`--mode vectorized`
- 减少回测时间范围
- 确保数据已缓存

## 10. 进阶开发

1. 在 `data/` 目录下创建新的数据源适配器

3. 在 `config/settings.py` 中配置新数据源

### 10.3 自定义指标

1. 在 `utils/metrics.py` 中添加新的性能指标
2. 在回测结果中使用新指标
