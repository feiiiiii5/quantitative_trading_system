# 行为准则

## 我们的承诺

为了营造一个开放、友好的社区环境，我们承诺让所有参与者都有一个无骚扰的体验，无论年龄、体型、残疾、种族、性别特征、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

我们承诺以善解人意、以尊重的方式行事，避免挑剔挖苦的人身攻击。

## 我们的标准

有助于积极环境的行为示例：

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 与社区其他成员表现出同理心

不可接受的行为示例：

- 使用性语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬损评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 在专业环境中被认为不合适的行为

## 我们的责任

社区维护者有责任明确和公平地解释可接受行为的标准，并有责任采取适当和公平的纠正措施来应对任何不可接受的行为。

社区维护者有权和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并应适当地传达删除或编辑的理由。

## 范围

当一个人代表项目或其社区时，本行为准则同时适用于项目空间和公共空间。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发帖，或在在线或离线活动中担任指定代表。

## 强制执行

如有骚扰、跟踪或其他不可接受的行为，请通过以下方式联系社区管理团队：support@quantsystem.pro。

所有投诉都将得到审查和调查，并将做出必要和适当地回应。团队有义务对事件保密。具体执行政策的更多细节可以单独发布。

如果社区维护者不诚实或参与违反本行为准则的行为，其他社区领导人可以采取他们认为适当的所有措施，包括暂时禁止或永久开除。

## 处罚

对违反本行为准则的人可能采取的处罚措施包括：

1. **口头警告** - 口头警告，说明违规性质并要求停止不当行为
2. **书面警告** - 书面警告，记录违规行为和未来预期
3. **临时禁止** - 暂时禁止参与项目社区活动
4. **永久禁止** - 永久禁止参与项目社区活动
